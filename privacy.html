Personvern

Økonomisk fremtid er en lokal PWA. Appen oppretter ingen brukerkonto og sender ingen økonomiske opplysninger til en server.

Opplysninger lagres i nettleserens lokale lagring på enheten din. Du kan eksportere, importere eller slette data fra appens innstillinger.

Dersom appen publiseres via GitHub Pages, leveres kildekoden og statiske filer av GitHub. Personlige data som skrives inn i appen sendes ikke til GitHub av appen.
